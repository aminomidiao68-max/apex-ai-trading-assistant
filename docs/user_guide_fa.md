# راهنمای کاربر APEX AI

## ورود
- از اکانت دمو یا اکانت ثبت‌نامی خودت وارد شو.
- بعد از ورود، وارد داشبورد اصلی می‌شوی.

## داشبورد
- سشن جاری بازار را نشان می‌دهد.
- واچ‌لیست و استریم قیمت دارد.
- خلاصه ژورنال معامله را نمایش می‌دهد.

## Signals
- اسکن زنده انجام بده.
- سیگنال‌ها در history ذخیره می‌شوند.
- می‌توانی سیگنال را به ژورنال اضافه کنی.

## Chart
- نمودار کندلی را ببین.
- با drag افقی جابه‌جا شو.
- با gesture / دکمه‌ها zoom کن.
- با tap اطلاعات کندل را ببین.

## Journal
- معاملات باز و بسته را مشاهده کن.
- وضعیت سود/ضرر و آمار را بررسی کن.

## Backtest Lab
- بک‌تست عادی اجرا کن.
- sweep اجرا کن.
- walk-forward اجرا کن.
- خروجی نتایج را تحلیل کن.

## Broker
- وضعیت connectorها را ببین.
- preview سفارش بگیر.
- dry-run execution انجام بده.

## Profile / Settings
- ثبت device برای push
- اجرای تست notification
- تنظیمات UX و ریسک

## هشدار
قبل از استفاده واقعی:
- حتماً از testnet / practice استفاده کن
- تنظیمات ریسک را بررسی کن
- به نتایج بک‌تست به‌تنهایی تکیه نکن
