الان از تو دقیقاً چی می‌خوام؟

فایل Firebase را من قبلاً داخل پروژه گذاشتم. تو دیگر لازم نیست فایل بسازی.

الان فقط باید روی کامپیوتر / لپ‌تاپ پروژه را در Android Studio باز کنی و ببینی اپ بدون خطا اجرا می‌شود یا نه.

خیلی مهم

این مراحل با گوشی انجام نمی‌شود. برای ادامه باید یکی از این‌ها را داشته باشی:

  • لپ‌تاپ ویندوز
  • مک‌بوک
  • کامپیوتر رومیزی

اگر فقط گوشی داری، همین حالا به من بگو: «من فقط گوشی دارم» تا مسیر ساده‌تر پیشنهاد بدهم.

چک‌لیست خیلی کوتاه

  • Android Studio نصب باشد
  • پوشه پروژه روی کامپیوترت باشد
  • من فایل Firebase را داخل این مسیر گذاشته‌ام:
project/android/app/google-services.json
1Android Studio را باز کن

اگر روی کامپیوتر Android Studio نصب نیست، اول آن را نصب کن. اگر نصب است، بازش کن.

2پروژه اندروید را باز کن

در Android Studio این کار را انجام بده:

  • روی Open بزن
  • این پوشه را انتخاب کن:
project/android

دقت کن کل پوشه android را باز کنی، نه پوشه app را.

3صبر کن پروژه لود شود

بعد از باز شدن پروژه، Android Studio چند ثانیه یا چند دقیقه مشغول می‌شود.

اگر پایین صفحه چیزی مثل Gradle Sync دیدی، صبر کن تمام شود.

4اگر دکمه Sync دیدی، رویش بزن

گاهی بالای برنامه یک نوار زرد/آبی می‌آید و می‌گوید:

Sync Now

اگر دیدی، روی آن بزن.

5اگر خطا آمد، فقط اسکرین‌شات بفرست

اگر هر خطای قرمز یا زرد دیدی:

  • هیچ چیزی را حدس نزن
  • فقط اسکرین‌شات بگیر
  • برای من بفرست

من از روی اسکرین‌شات دقیقاً می‌گویم چه کار کنی.

6اگر Sync موفق بود، اپ را اجرا کن

بالای Android Studio معمولاً یک دکمه سبز مثل این هست:

▶ Run

روی آن بزن.

اگر ازت دستگاه خواست، یکی از این دو را انتخاب کن:

  • Emulator
  • گوشی واقعی با USB Debugging
7نتیجه‌ای که باید ببینی
  • اول عکس برند APEX AI نمایش داده می‌شود
  • بعد صفحه ورود می‌آید
  • اگر اینجا رسیدی یعنی بخش اندروید تقریباً درست است
8ورود آزمایشی

اگر اپ باز شد، با این مشخصات وارد شو:

  • Email: demo@apexai.app
  • Password: Demo12345!
9بعد از ورود برو به Profile

در Profile دو دکمه مهم هست:

  • Register Device for Push
  • Send Test Push Event

فعلاً فقط اگر اپ باز شد، همین مرحله کافی است. لازم نیست همه چیز را یک‌باره تست کنی.

اگر بخواهم خیلی ساده بگویم

  1. Android Studio را باز کن
  2. پوشه project/android را باز کن
  3. اگر لازم بود Sync Now را بزن
  4. اگر خطا آمد، اسکرین‌شات برای من بفرست
  5. اگر خطا نیامد، دکمه ▶ Run را بزن
