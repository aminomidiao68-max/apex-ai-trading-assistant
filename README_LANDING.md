## What is APEX AI?

APEX AI is a product-ready foundation for a mobile trading assistant that combines:
- smart-money-inspired signal workflows
- structured risk management
- execution preview and broker foundations
- live monitoring
- chart interaction
- backtesting and walk-forward analysis

It is designed as a **serious MVP / product foundation**, not a toy demo.

## Getting Started

### Backend
```bash
cd project/backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Android
Open:
```text
project/android
```

Demo account:
- `demo@apexai.app`
- `Demo12345!`
